# Plants vs. Zombies Heroes - Random Number Generator

Here is a list of available commands:
- `/draw`
- `/draw <count>`
- `/conjure <key:value...>`
- `/conjure <key:value...> <count>`
- `/ping`
- `/block`
- `/shuffle <card> <copies>`
- `/start`
- `/randint <max>`
